# Apna PG – PG Accommodation Platform

Apna PG is a responsive web application developed to simplify the process of finding
Paying Guest (PG) accommodations. The platform allows users to browse available PGs,
search based on requirements, and contact owners easily.

## Overview
The project focuses on providing a clean user interface and essential features to
improve accessibility and user experience for students and working professionals
looking for PG accommodations.

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- MongoDB

## Features
- PG accommodation listings
- Search functionality for easy discovery
- Contact inquiry form for user interaction
- Responsive design for mobile and desktop devices

## How to Run the Project
1. Clone the repository:
   git clone https://github.com/sanskruti1704/Apna_PG
2. Install dependencies:
   npm install
3. Start the server:
   npm start
4. Open the application in a web browser.
